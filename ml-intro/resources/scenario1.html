Red-Green Scenario

The Red-Green Scenario demonstrates a simple neural network that predicts one of two outputs: red or green. The neural network takes a 2D input corresponding to a selected color and learns to classify it as either "red" or "green".

Key Features

  • Input: Users can select a color on the color picker and see the normalized RGB input values.
  • Output: The neural network predicts one of two classes: red or green.
  • Training Parameters: Bias, learning rate, and activation functions for the output layer can be adjusted.
  • Training: The network can be trained for a chosen number of epochs using the Run button.
  • Save/Load: Save the current state of the network or load a previously saved state.

Recommendations for Training

  • Start with a bias value around 1.0 for balanced predictions, then adjust based on performance.
  • Choose an appropriate activation function:
    • Sigmoid: Suitable for binary classification tasks.
    • Tanh: Normalizes the output between -1 and 1, often useful for hidden layers.
  • Train the network for at least 10 epochs initially to observe the learning trend.
  • Periodically save the model to preserve progress.
  • If predictions are skewed, consider adjusting the bias or introducing regularization.

How to Use

  1. Select a color from the color picker.
  2. Choose the output class (red or green).
  3. Adjust the bias, learning rate, and activation function as needed.
  4. Set the number of training epochs.
  5. Click Run to train the network.
  6. Save the trained model using the Save button.
  7. To reload a saved model, click Load.
